# General Tools

## Making HTTP requests

[Postman](https://www.getpostman.com/) is a great tool for testing APIs. It is available as a Chrome app or as a native app for Mac, Windows, and Linux.

[HTTPie](https://httpie.org/) is a command line tool for making HTTP requests. It is available for Mac, Windows, and Linux.

## SQL managing

[DBngin](https://dbngin.com/) is a free tool for managing MySQL, PostgreSQL, and Redis databases on Mac.

[TablePlus](https://tableplus.io/) is a free tool for managing MySQL, PostgreSQL, and Redis databases on Mac.
