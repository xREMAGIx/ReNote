
# Markdown Cheatsheet

## Basic Syntax
| Element | Markdown Syntax |
| --- | --- |
| **Heading** | # H1 <br/> ## H2 <br/> ### H3 <br/> #### H4 <br/> ##### H5 <br/> ###### H6 |
| **Bold** | `**bold text**` |
| **Italic** | `*italicized text*` |
| **Strikethrough** | `~~strikethrough text~~` |
| **Blockquote**| `> blockquote` |
| **Code** | \`code` |
| **Horizontal Rule** | `---` |
| **Link** | `[title](https://www.example.com)` |
| **Image** | `![alt text](image.jpg)` |
